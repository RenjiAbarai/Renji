

# 路由器介绍



### 网络连接模式

- AP(接入点)模式

- Router(无线路由)模式

- Repeater(中继)模式

- Bridge(桥接)模式

- Client(客户端)模式

  

|             模式             |  路由器   | 中继 |    AP     | 桥接 |
| :--------------------------: | :-------: | :--: | :-------: | :--: |
|            防火墙            |    有     |  无  |    无     |  无  |
|        与上级路由网段        |   不同    |  同  |    同     |  同  |
| 下级路由访问上级路由连接设备 |   支持    | 支持 |   支持    | 支持 |
| 上级路由访问下级路由连接设备 |  不支持   | 支持 |   支持    | 支持 |
|         信号拓展方式         |   有线    | 无线 |   有线    | 无线 |
|         设备接入方式         | 有线/无线 | 无线 | 有线/无线 | 有线 |
